README


El proyecto es el desarrollo del primer proyecto individual de Henry. El proyecto se desarrolla por medio de un archivo de películas el cual tiene entre otras cosas, rating, título, una pequeña descripción y el cast de las películas. El proyecto  se propone un procesamiento de datos, un análisis de datos y unas posibles aplicaciones del manejo de los datos además de un sistema de recomendaciones. Por lo cual puede ser útil para aquellos interesados en el procesamiento de datos, en el análisis de datos y en NPL y en la aplicación de los mismos.


El siguiente repo contiene los siguientes archivos:


main.py: aquí se tienen las 7 funciones para consulta. Es necesario tener pandas y ScikitLearn.


mach_learn_proyecto_uno_henry: aquí se tiene el trabajo de EDA Y ML en él se analizan los datasets que nos dio Henry dichos datos fueron muy bien analizados por la biblioteca: missigno, autoviz y matplo (los cuales es necesario instalar con un pip) t, y  además de que se plantean diferentes parámetros para el modelo de machine learning: Dicho modelo se puede seguir desarrollando pero por falta de computo se deja de una manera más simple. El código se corrió sobre colab por lo que se puede usar en dicho ambiente. De este archivo se extrajo el archivo para el modelo por medio de filtros.


pro_inv_uno_henry.ipynb.:En este archivo se hace el preprocesamiento de las bases de datos el cual esta en un formato de celdas de colab. 


data_trans: es un csv comprimido y preprocesado del que nos proporcionó Henry y es necesario para que las funciones corran.


df_machler.xlsx: es un archivo excel el cual es un un filtrado de el dataset original y es necesario para que la función que tiene el modelo de machine learning funcione.


requirements.txt: es el conjunto de librerías necesarias para poder hacer el deployment.


archi_chiquito.xlsx: es un excel para otra posible configuración del modelo.
